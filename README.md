## ✨ 주요 기능

이 프로젝트에서 구현한 핵심 기능입니다.

* **To-Do List:** 할 일을 추가하고, 목록에서 삭제할 수 있습니다. (`localStorage`에 저장)
* **실시간 날씨:** OpenWeatherMap API를 `fetch`로 호출하여 현재 위치의 날씨와 온도를 표시합니다.
* **유저 이름 저장:** 유저의 이름을 입력받아 `localStorage`에 저장하고, 환영 메시지를 띄웁니다.
* **랜덤 배경화면:** 새로고침 시마다 랜덤으로 배경 이미지가 변경됩니다.
* **오늘의 명언:** 준비된 명언 중 하나를 랜덤으로 표시합니다.

<br>

## 💻 사용 기술

* HTML5
* CSS3
* **Vanilla JavaScript (ES6+)**

<br>

## 💡 이 프로젝트를 통해 배운 점

* JavaScript를 이용한 **DOM 요소 선택, 생성, 제어** 방법
* **`localStorage`**를 활용해 브라우저에 데이터를 저장하고 불러오는 방법
* `fetch`를 사용한 **Open API 호출** 및 비동기 처리 (Promise)
* `Math.random()` 등 내장 객체를 활용한 랜덤 기능 구현
